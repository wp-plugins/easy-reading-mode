== Description ==

This plugin lets your website user read the main content of your website in an easy distraction free reading mode.
This plugin works in the same way as "Reader" in Safari browser.

This plugin adds a button "Reading Mode" before the post content and when clicked on it, a popup window appears only with the post title and content.

== Installation ==

1. Download the zip file of this plugin
2. Go to Plugins -> Add New on the wordpress backend. Click on "Upload" option which appears on the top.
3. Select the plugin file and upload. Click on "Activate" on the next page that appears.

To upload the plugin via FTP follow these steps:

1. Extract the zip file of the plugin.
2. Upload the plugin folder in the "wp-content/plugins" folder of your wordpress via FTP.
3. Go to "Installed Plugins" page on your backend and click on "Activate".
